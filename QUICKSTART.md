# Quick Start Guide 🚀

Get your Personal Finance Tracker up and running in 5 minutes!

## Step 1: Install Dependencies

```bash
npm install
```

## Step 2: Start Development Server

```bash
npm run dev
```

## Step 3: Open in Browser

Navigate to [http://localhost:3000](http://localhost:3000)

## Step 4: Start Tracking!

### Add Your First Income
1. Click **Income** in the sidebar
2. Click **Add Income** button
3. Fill in the details:
   - Category: Salary
   - Amount: 50000
   - Date: Today's date
   - Notes: Monthly salary
4. Click **Add Income**

### Add Your First Expense
1. Click **Expenses** in the sidebar
2. Click **Add Expense** button
3. Fill in the details:
   - Category: Food
   - Amount: 5000
   - Date: Today's date
   - Notes: Groceries
4. Click **Add Expense**

### Add Your First Investment
1. Click **Investments** in the sidebar
2. Click **Add Investment** button
3. Fill in the details:
   - Category: Stocks
   - Amount: 10000
   - Date: Today's date
   - Notes: Tech stocks
4. Click **Add Investment**

## Step 5: Explore Features

### View Dashboard
- Click **Dashboard** to see your financial overview
- Check total income, expenses, savings, and net worth
- View recent transactions

### Check Analytics
- Click **Analytics** to see visual charts
- View income vs expenses trends
- See category breakdowns

### Generate Reports
- Click **Monthly Reports** to see current month summary
- Click **Yearly Reports** to see annual overview
- Export reports as PDF

### Filter Transactions
- Click **Transactions** to see all entries
- Use filters to find specific transactions
- Export data as CSV

### Toggle Dark Mode
- Click the theme toggle at the bottom of sidebar
- Switch between light and dark modes

## Common Tasks

### Edit a Transaction
1. Go to the relevant page (Income/Expenses/Investments)
2. Find the transaction
3. Click the edit icon (pencil)
4. Make changes
5. Click **Update**

### Delete a Transaction
1. Go to the relevant page
2. Find the transaction
3. Click the delete icon (trash)
4. Confirm deletion

### Export Data
**CSV Export:**
- Go to Transactions page
- Click **Export CSV**

**PDF Export:**
- Go to Monthly or Yearly Reports
- Click **Export PDF**

## Tips for Best Results

1. **Be Consistent**: Add transactions regularly
2. **Use Categories**: Properly categorize for better insights
3. **Add Notes**: Include details for future reference
4. **Review Monthly**: Check reports at month-end
5. **Backup Data**: Export CSV regularly
